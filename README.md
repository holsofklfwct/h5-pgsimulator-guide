# h5-pgsimulator-guide

## 项目简介

本仓库用于归档与发布多个独立的 HTML 页面。这些页面以模拟器或指南类工具为主，适合在浏览器中直接打开使用，不依赖后端服务。仓库本身不针对任何特定域名或网站，仅作为静态页面的存储与分发入口。

## 目录说明

```
h5-pgsimulator-guide/
├── index.html         # 仓库入口页面，列出所有归档页面链接
├── pages/             # 存放独立 HTML 页面的文件夹
│   ├── simulator-a.html
│   ├── guide-b.html
│   └── ...
└── assets/            # 公共资源（样式、脚本、图片等）
    ├── css/
    ├── js/
    └── images/
```

- **pages/**：每个 HTML 文件均为独立功能页面，可直接在浏览器中运行。
- **assets/**：存放页面间共享的静态资源，便于统一维护。

## 页面归档说明

每个页面均以独立的 HTML 文件形式存在，内容聚焦于模拟或引导功能，不包含任何外部跟踪、广告或跳转逻辑。页面的设计遵循通用 Web 标准，兼容主流现代浏览器。

如需新增页面，请将文件放入 `pages/` 目录，并在 `index.html` 中添加相应链接。建议为每个页面提供简洁的标题与功能描述，方便查阅。

## 维护说明

- 本仓库由维护者不定期更新，新增页面或修正现有内容。
- 页面内容保持轻量、自包含，避免引入过多外部依赖。
- 欢迎提交 Issue 或 Pull Request 进行改进，但请注意保持仓库的通用性与简洁性。
- 所有页面均为静态资源，部署后可直接通过 GitHub Pages 或其他静态托管服务访问。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中打开 `index.html`，即可查看所有归档页面列表。
3. 点击任意页面链接，即可在浏览器中直接使用对应功能。

无需配置服务器或数据库，完全离线可用。
